 What is the purpose of app.use() method in express js?

->app.use() in Express.js:
Purpose: Mounts middleware functions in the request processing pipeline.
Allows middleware to modify request and response objects or pass control to the next middleware.

 What is the purpose of  app.method()  method in express js?

->The app.METHOD() function is used to route an HTTP request, where METHOD is the HTTP method of the request, such as GET, PUT, POST, and so on, in lowercase

 How to use find(), findIndex() and filter() methods on Arrays in JavaScript? 

->find(): Finds the first element in an array that satisfies a provided testing function.

->findIndex(): Returns the index of the first element in an array that satisfies a provided testing function.

->filter(): Creates a new array with all elements that pass the test implemented by the provided function.
